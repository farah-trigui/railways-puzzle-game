Game Description

Map Setup :

The game takes place on a square grid, which varies in size depending on the difficulty:
Easy: 5x5 grid
Hard: 7x7 grid
Different types of terrain tiles appear on the map:
  • Empty Tile: Allows railway construction in any of the three directions, except for the entry direction.
  • Bridge: The railway can only be built in a straight line across the bridge.
  • Mountain: Blocks two exits, so the train can only turn 90Â°.
  • Oasis: Cannot support railway construction.

Building the Railway

Place railway tracks to connect tiles according to the terrain rules:
  • The railway must not cross over itself or loop back to any cell.
  • The train can only pass through each cell once.
Choose how you want to place and manage tracks:
  • Clicking on cells: Cycle through available track types.
  • Right-click options: Rotate or delete placed tracks.
  • Drag and Draw Hold down the mouse to draw tracks across cells.

Game Interface

When the game starts, the main menu appears:
  • Enter your Name: This will personalize your experience.
  • Select Map Difficulty: Choose between Easy and Hard.
  • You can also view the game description in the menu.

End of the Game

The game ends when the player correctly completes the puzzle according to the given rules. During the validation of the puzzle's completion, the following elements need to be checked:
  • Each cell has two exit points, and the train can enter each neighboring cell from the correct direction.
  • Every cell that needs to be touched is touched.
  • Every cell where no track can be placed remains empty.
